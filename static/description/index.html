Description

DebugPY for Visual Studio Code

Configuration

Server side you have to install debugpy via pip or pip3 pip install debugpy
One installed, there is no need to configure.
On your Visual Studio Code you have to define this server with IP address and Port 5678.

Setup

  1. Press CTRL + SHIFT + D or CMD + SHIFT + D
  2. Create file launch.json if not already present
  3. insert this code (change IP 10.0.0.1 according to your development server):

    {
        "version": "0.2.0",
        "configurations": [
            {
                "name": "Python: Debugpy",
                "type": "python",
                "request": "attach",
                "port": 5678,
                "host": "10.0.0.1",
            }
        ]
    }
More info on official page

How to use

On the code, create breakpoint simply by clicking beside the line number where we want to stop it. Then press F5 to start the debugger, go to Odoo and load the page you are looking for. When the python process goes to the line it stops. Here you can inspect variables, call stacks and so on.
